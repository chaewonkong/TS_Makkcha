### Installation

#### Install nvm, yarn

```bash
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash
$ command -v nvm
$ which node
$ npm install -g yarn
```



#### Install project

```bash
$ nvm install 10.16.0
$ nvm use
$ yarn install
$ cd ios && pod install
```




#### Start project

```bash
# iOS
$ react-native run-ios

# Android
$ react-native run-android
```
